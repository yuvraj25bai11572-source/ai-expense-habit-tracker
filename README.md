# 💰 AI Expense Tracker

## 📌 Problem

Managing daily expenses is a common challenge for students and individuals. People often lose track of where their money is spent, leading to poor financial planning.

## 💡 Solution

This project is a simple expense tracking system that allows users to:

* Record daily expenses
* View all expenses
* Analyze total spending
* Understand spending patterns by category

## 🚀 Features

* Add expenses with amount, category, and description
* Store data in CSV file
* View all recorded expenses
* Calculate total spending
* Category-wise expense analysis

## 🛠️ Tech Stack

* Python
* Pandas
* CSV (for data storage)

## ⚙️ Setup Instructions

1. Clone the repository:

   ```
   git clone https://github.com/YOUR_USERNAME/ai-expense-habit-tracker.git
   ```

2. Navigate to the project folder:

   ```
   cd ai-expense-habit-tracker
   ```

3. Install dependencies:

   ```
   pip install pandas
   ```

4. Run the application:

   ```
   python app.py
   ```

## 📊 Usage

1. Run the program
2. Choose an option:

   * Add Expense
   * View Expenses
3. Analyze your total and category-wise spending
